class Image: tkd.element.element.Element;

The image base class.

this();

Construct the image.

void blank();

Clears the image of all pixel data and effectively makes it transparent.

void setData(string data);

Set the image data. This should be base64 encoded binary data.

Parameters

string data Base64 encoded data.

string getData();

Get the image data.

Return Value

Base64 encoded image data.

void setFormat(string format);

Set the image format. Once set the image only accepts files or data in this format.

Parameters

string format A valid image format.

See Also

tkd.image.imageformat for supported formats.

string getFormat();

Get the image format.

Return Value

The image format.

void setFile(string file);

Set the file to read the image data from.

Parameters

string file The file to read the image data from.

void embedDataFromFile(string filename)();

This method embeds the image as a base64 encoded string into the application. The path to the image must be passed to the compiler using the -J switch.

Parameters

filename The filename to read the data from.

string getFile();

Get the image file.

Return Value

The file that was loaded for this image.

void setGamma(string gamma);

Set the image gamma.

Parameters

string gamma The destination gamma.

string getGamma();

Get the current image gamma.

Return Value

A string containing the image gamma setting.

void setHeight(int height);

Set the image height.

Parameters

int height The height to crop the image to.

string getHeight();

Get the current cropped image height.

Return Value

The current cropped image height. Returns 0 if no cropping is taking place.

void setPalette(string palette);

Set the image palette. This setting set how many levels of color/gray are used.

Sample Palettes:

  • "16" = 16 levels of gray
  • "16/16/16" = 16 levels of RGB

Parameters

string palette A string describing the palette to use.

string getPalette();

Get the current palette.

Return Value

A string describing the palette.

void setWidth(int width);

Set the image width.

Parameters

int width The width to crop the image to.

string getWidth();

Get the current cropped image height.

Return Value

The current cropped image height. Returns 0 if no cropping is taking place.

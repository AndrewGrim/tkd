class Logger;

A simple class to provide logging support.

nothrow this(string logFile = null);

Constructor.

If a log file is not passed, log instead to stdout.

Parameters

string logFile The log file for logging.

nothrow void eval(A...)(string text, A args);

Write eval text to the log.

Parameters

string text The format of the text to write to the log.
A args The arguments that the format defines (if any).

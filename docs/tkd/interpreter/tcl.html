class Tcl;

Simple wrapper for the Tcl interpreter.

static Tcl getInstance();

Get an instance of this class.

Return Value

If An instance doesn't exist, one is created and returned. If one already exists, that is returned.

string[] escape(string[] args);

Escape harmful characters in arguments that are to be used in a script.

Parameters

string[] argsAn array of arguments to escape.

Return Value

Escaped arguments.

string escape(string arg);

Escape harmful characters in the passed argument that is to be used in a script.

Parameters

string argThe argument to escape.

Return Value

The argument.

void eval(A...)(string script, A args);

Evaluate a script fragment using the interpreter.

Parameters

string scriptThe script to evaluate, including any format placeholders.
A argsvariadic list of arguments to provide data for any format placeholders.

void setResult(A...)(string result, A args);

Set the result of the interpreter. This is sometimes used to set the result to an error if things go bad.

Parameters

string resultThe text to set as the result, including any format placeholders.
A argsvariadic list of arguments to provide data for any format placeholders.

T getResult(T)();

Get the result string from the interpreter.

Return Value

A string representing the result of the last script fragment evaluated.

Tcl_Command createCommand(string name, Tcl_CmdProc commandProcedure, ClientData data = null, Tcl_CmdDeleteProc deleteProcedure = null);

Create a new command in the Tcl interpreter.

Parameters

string nameThe name of the new command.
Tcl_CmdProc commandProcedureA function pointer to the new command.
ClientData dataExtra data to be passed to the command on invocation.
Tcl_CmdDeleteProc deleteProcedureThe procedure to run when deleteCommand is called.

Return Value

A command token that can be used to refer to the command created.

void deleteCommand(string name);

Delete a command in the Tcl interpreter.

Parameters

string nameThe name of the command to delete.

void setVariable(T)(string name, T value);

Set the value of a variable. If the variable doesn't exist it is created.

Parameters

string nameThe name of the variable to set.
T valueThe variable's value.

string getVariable(string name);

Get the value of a variable.

Parameters

string nameThe name of the variable to get the value of.

Return Value

A string containing the variable's value.

void deleteVariable(string name);

Delete a variable from the interpreter.

Parameters

string nameThe name of the variable to delete.

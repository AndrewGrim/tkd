class Tcl;

Simple wrapper for the Tcl interpreter.

protected Tcl_Interp* _interpreter;

An instance of the native tcl interpreter.

protected Logger _log;

The logger.

protected this();

Create the interpreter and initialise it.

Exceptions Thrown

Exception if Tcl interpreter cannot be initialised.

static Tcl getInstance();

Get an instance of this class.

Return Value

If An instance doesn't exist, one is created and returned. If one already exists, that is returned.

void eval(A...)(string script, A args);

Evaluate a script fragment using the interpreter.

Parameters

string script The script to evaluate, including any format placeholders.
A args variadic list of arguments to provide data for any format placeholders.

Exceptions Thrown

Exception if the script evaluation failed.

void setResult(A...)(string result, A args);

Set the result of the interpreter. This is sometimes used to set the result to an error if things go bad.

Parameters

string result The text to set as the result, including any format placeholders.
A args variadic list of arguments to provide data for any format placeholders.

T getResult(T)();

Get the result string from the interpreter.

Return Value

A string representing the result of the last script fragment evaluated.

Tcl_Command createCommand(string name, Tcl_CmdProc commandProcedure, ClientData data = null, Tcl_CmdDeleteProc deleteProcedure = null);

Create a new command in the Tcl interpreter.

Parameters

string name The name of the new command.
Tcl_CmdProc commandProcedure A function pointer to the new command.
ClientData data Extra data to be passed to the command on invocation.
Tcl_CmdDeleteProc deleteProcedure The procedure to run when deleteCommand is called.

Return Value

A command token that can be used to refer to the command created.

void deleteCommand(string name);

Delete a command in the Tcl interpreter.

Parameters

string name The name of the command to delete.

Exceptions Thrown

Exception if the command cannot be deleted.

void setVariable(string name, string value);

Set the value of a variable. If the variable doesn't exist it is created.

Parameters

string name The name of the variable to set.
string value The variable's value.

string getVariable(string name);

Get the value of a variable.

Parameters

string name The name of the variable to get the value of.

Return Value

A string containing the variable's value.

void deleteVariable(string name);

Delete a variable from the interpreter.

Parameters

string name The name of the variable to delete.

Exceptions Thrown

Exception if the command cannot be deleted.

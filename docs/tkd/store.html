struct Store(T);

The store struct.

T[string] _items;

Storage for all items.

void opDispatch(string name)(T item, string file = __FILE__, size_t line = __LINE__);

Handle storing items as properties of this struct. If a property doesn't exist when assigning a item to it, this method is called.

Parameters

name The property name of the accessed item.
T item The item assigned to the property name.
string file The file name where this action occurred for error reporting.
size_t line The line number where this action occurred for error reporting.

See Also

http://dlang.org/operatoroverloading.html#Dispatch

T opDispatch(string name)(string file = __FILE__, size_t line = __LINE__);

Handle retrieving items as properties of this struct.

Parameters

name The property name of the accessed item.
string file The file name where this action occurred for error reporting.
size_t line The line number where this action occurred for error reporting.

Return Value

The item matching the property name.

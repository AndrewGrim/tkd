auto pack(this T)(int outerPadding = 0, int innerPadding = 0, string side = GeometrySide.top, string fill = GeometryFill.none, string anchor = AnchorPosition.center, bool expand = false);

Geometry method for placing this widget inside its parent. If no parent was specified the main window is used as a parent.

Parameters

int outerPadding The amound of padding to add around the widget.
int innerPadding The amound of padding to add inside the widget.
string side The side to place the widget inside its parent.
string fill The space to fill inside its parent.
string anchor The anchor position of the widget inside its parent.
bool expand Whether or not to expand to fill the entire given space.

Return Value

This widget to aid method chaining.

enum GeometrySide: string;

Side values for widget geometry layout.

left

Set geometry to the left.

right

Set geometry to the right.

top

Set geometry to the top.

bottom

Set geometry to the bottom.

enum GeometryFill: string;

Fill values for widget geometry layout.

none

No filling.

x

Fill the available horizontal space.

y

Fill the available vertical space.

both

Fill all available space.

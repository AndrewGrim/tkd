void addTab(string text, Widget widget);

Add a tab to the notebook. When adding a tab to the notebook the tab gains an id that is equal to the passed widget's id and can be used later to refer to the new tab.

Parameters

string text The text of the tab.
Widget widget The widget to add as the tab pane.

void insertTab(int tabIndex, string text, Widget widget);

Insert a tab into the notebook at a specified zero based index. When adding a tab to the notebook the tab gains an id that is equal to the passed widget's id and can be used later to refer to the new tab. If the id of the widget passed is already used as a tab id then that existing one will be moved to the new position.

Parameters

int tabIndex The index where the new tab will be created.
string text The text of the tab.
Widget widget The widget to add as the tab pane.

void insertTab(string tabId, string text, Widget widget);

Insert a tab into the notebook before the specified tabId. When adding a tab to the notebook the tab gains an id that is equal to the passed widget's id and can be used later to refer to the new tab. If the id of the widget passed is already used as a tab id then that existing one will be moved to the new position.

Parameters

string tabId The tab id to position the tab before.
string text The text of the tab.
Widget widget The widget to add as the tab pane.

void removeTab(int tabIndex);

Remove a tab from the notbook via its index.

Parameters

int tabIndex The zero based index of the tab to remove.

void removeTab(string tabId);

Remove a tab from the notbook via its string id. The string id is the widget id that was added as the tab.

Parameters

string tabId The tab string id of the tab to remove.

void hideTab(int tabIndex);

Hide a tab from the notbook via its index.

Parameters

int tabIndex The zero based index of the tab to hide.

void hideTab(string tabId);

Hide a tab from the notbook via its string id. The string id is the widget id that was added as the tab.

Parameters

string tabId The tab string id of the tab to hide.

int getTabIndexById(string tabId);

Get the tab index from its string id. The string id is the widget id that was added as the tab.

Parameters

string tabId The tab string id of the tab.

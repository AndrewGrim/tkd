enum State: string;

State values of widgets.

normal

Normal state.

active

Active state.

disabled

Disabled state.

focus

Focused state.

pressed

Pressed state.

selected

Selected state.

background

Background state.

readonly

Readonly state.

alternate

Alternate state.

invalid

Invalid state.

hover

Hover state.

hidden

Hidden state. Only applies to canvas widgets.

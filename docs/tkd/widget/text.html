class Text: tkd.widget.widget.Widget, tkd.widget.common.xscrollcommand.IXScrollable!(Text).IXScrollable, tkd.widget.common.yscrollcommand.IYScrollable!(Text).IYScrollable;

A text widget displays one or more lines of text and allows that text to be edited. Text widgets support embedded widgets or embedded images.

Example:

auto text = new Text()
	.appendText("Text")
	.pack();

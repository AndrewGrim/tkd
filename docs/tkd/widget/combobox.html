class ComboBox: tkd.widget.widget.Widget;

Class representing a combo box widget.

This widget has two types of values that can be set. First, a list of values can be set to populate the drop-down list which can then be selected via a mouse. Second, the value can be set independently and in addition to the value list. See the below widget specific and common commands for an overview of how this works.

Additional Events:

Widget specific events that can be bound to using the bind command.

<<ComboboxSelected>>The combobox widget generates this event when the user selects an element from the list of values.

tkd.element.UiElement

int getSelected();

Get the index of the selected value from the current widget's list values.

Return Value

The index of the selected list value. Indexes start at 0 and -1 is returned if the current value does not appear in the list.

void select(int index);

Select the value at a particular index in the value list.

Parameters

int index The index of the value to select.

string getValue(T = string)();

Get the selected value.

Parameters

T The type of the value to return.

Return Value

The value of the widget.

void setValue(string value);

This method has two uses. First it can change the value independently of the existing value list. Second, if setting a value from the value list, it sets the selected index to that of the position in the list. The getSelected method above then returns the index.

Parameters

string value The new widget value.

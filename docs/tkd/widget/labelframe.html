void setLabel(Label label);

Set the label widget to use for the label. The label must be a child of the labelframe widget or one of the labelframe's ancestors, and must belong to the same top-level widget as the labelframe. If set, overrides the text parameter.

Parameters

Label label The label widget to use as the label.

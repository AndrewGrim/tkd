template Selection()

These are common commands that apply to all widgets that have them injected.

auto deselectText();

Clear the text selection.

Return Value

This widget to aid method chaining.

bool isTextSelected();

Check if any text selected.

auto selectText(int startCharIndex = 0, int endCharIndex = (int).max);

Select the text.

Parameters

int startCharIndex The index where the selection starts.
int endCharIndex The index where the selection ends.

Return Value

This widget to aid method chaining.

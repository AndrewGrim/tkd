template OutlineWidth()

These are common commands that apply to all widgets that have them injected.

int getOutlineWidth();

Get the width of the outline.

Return Value

The width of the outline;

auto setOutlineWidth(this T)(int width);

Set the width of the outline.

Parameters

int width The width of the outline.

Return Value

This widget to aid method chaining.

int getActiveOutlineWidth();

Get the width of the active outline. An item's active state is triggered when the mouse rolls over the item.

Return Value

The width of the active outline;

auto setActiveOutlineWidth(this T)(int width);

Set the width of the active outline. An item's active state is triggered when the mouse rolls over the item.

Parameters

int width The width of the active outline.

Return Value

This widget to aid method chaining.

int getDisabledOutlineWidth();

Get the width of the disabled outline.

Return Value

The width of the disabled outline;

auto setDisabledOutlineWidth(this T)(int width);

Set the width of the disabled outline.

Parameters

int width The width of the disabled outline.

Return Value

This widget to aid method chaining.

Bugs

template Command()

These are common commands that apply to all widgets that have them injected.

auto setCommand(this T)(CommandCallback callback);

Add a command to a widget.

Parameters

CommandCallback callback The delegate callback to execute when invoking the command.

Return Value

This widget to aid method chaining.

Callback Arguments:

These are the fields within the callback's CommandArgs parameter which are populated by this method when the callback is executed.

CommandArgs.elementThe element that executed the callback.
CommandArgs.callbackThe callback which was executed.
